# Project: Ongudai Camp

## Vision
A modern travel booking platform for Altai region (Ongudai), focusing on hotels, tours, and activities with high-end UI/UX and localization.

## Core Goals
- [ ] Provide seamless booking experience for hotels and tours.
- [ ] Implement AI-driven chat support for travelers.
- [ ] Support multiple languages (RU, EN, TR).
- [ ] Integrated SMS verification for secure authentication.

## Tech Stack
- **Framework**: Next.js 16 (App Router, Turbopack)
- **Database**: SQLite (local) / Supabase (planned)
- **ORM**: Prisma 5.22.0
- **Auth**: NextAuth.js (v5 beta) with Social (VK, Telegram, Yandex) and Phone (SMS) support.
- **Styling**: Tailwind CSS 4, SaaS UI Master principles.
- **i18n**: next-intl (RU, EN, TR).
- **Animations**: Remotion (planned for showcases).
- **Data Fetching**: TanStack Query (v5+ installed).
