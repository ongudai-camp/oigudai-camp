# Integrations

## Current
- **Prisma**: Connected to local SQLite (`dev.db`).
- **NextAuth.js**: Configured with Social (VK, Telegram, Yandex) and Phone (SMS).
- **next-intl**: 3 locales (ru, en, tr) with geolocation detection in `proxy.ts`.
- **Remotion**: Integrated with a Property Showcase page in the admin dashboard.
- **TanStack Query**: Installed, pending refactor of data fetching logic.

## Planned
- **Supabase**: Target for production database and possibly storage.
- **SMS Service**: Twilio or SMS.ru for real phone verification.
- **AI API**: OpenAI or Gemini for the support chat system.
